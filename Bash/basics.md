# Basics

## Navigation

Print current directory:

```bash
pwd
```
List files and folders in current directory:

```bash
ls
```

Change directory:

```bash
cd path/to/file
```

Read an entire file:

```bash
cat path/to/file
```

Read a file bit by bit:

```bash
less path/to/file
```

Use the following commands with `less`:

- `Enter`: Jump to next line
- `Space`: Jump to next page
- `q`: Quit
