# Contributing

Thank you for wanting to contribute to my repository.

I welcome contributions that fix typos and correct wrong, misleading or imprecise content. You can contribute by forking the repository, make a commit with a clear and concise description and submit it as a pull request.

If you want to contribute your own TIL content, I suggest you start your own repository as this repository is intended as a personal learning journey.

Feel free to open an issue if you have any questions or suggestions.
